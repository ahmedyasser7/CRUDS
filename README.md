# CRUDS

A simple CRUD (Create, Read, Update, Delete) application for managing data efficiently.

## Features

- **Create** - Add new records
- **Read** - View existing records
- **Update** - Modify record details
- **Delete** - Remove records

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/ahmedyasser7/CRUDS.git
cd CRUDS
npm install
```

### Usage

```bash
npm start
```

## Technologies

- Vanilla JavaScript
- HTML/CSS
